## What this portfolio does

The portfolio is a single HTML file (`index.html`) that includes:

- A **hero section** with a terminal animation showing the developer's tech stack at a glance
- An **about section** with dynamic project counters
- A **skills section** with animated progress bars for C# .NET (98%), SQL Server (95%), and JavaScript (90%)
- A **projects section** where visitors can dynamically add project cards — persisted in memory during the session
- A **contact section** with direct email and LinkedIn links
- A **dark/light theme toggle** that persists user preference via `localStorage`
